## ChatBot

A full stack Chat Bot for question answering with features like data streaming and resume chat.

## Description

A Full stack web application built using python (langgraph, fastapi, pymongo) and javascript (React, express, mongodb). Backend has a basic microservice implementation, it have services like auth, chat and frontend-backend communicates via gateway. Application provides response quickly in the form of data stream and users can store thier chat history and resume the chat anytime.

## key features

-   **Question Anwsering :** User can ask any question,and LLM will provide the response.
-   **Data Streaming :** Response are provided as soon as it is generated in the form of data stream.
-   **Resume Chat :** Multiple chats can be created and each conversation history are stored in the database.
-   **Authentication :** Google OAuth and JWT are used for authentication.

## Technologies Used

-   **Frontend :** React, RTK, axios, tailwind css
-   **Backend :**
    -   `python` : Langgraph, FastAPi, pymongo
    -   `javascript` : NodeJS, Express, JWT, Mongoose, passport-google-oauth20
-   **Database :** MongoDB, SQLlite
-   **Containerization :** Docker
